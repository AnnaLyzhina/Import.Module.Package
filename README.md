# Бухгалтерия

Домашнее задание по теме `Import. Module. Package`.

## Структура проекта

- `main.py`
- `dirty_main.py`
- `application/salary.py`
- `application/db/people.py`
- `requirements.txt`

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

## Что делает проект

- выводит текущую дату и время;
- вызывает функцию получения сотрудников;
- демонстрирует простой расчёт зарплаты с помощью `pandas`.
